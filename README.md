# good_neighbor
A student project by cybersecurity students aimed at digitizing Ukrainian HOA. 'Good Neighbor' is a secure management platform focused on solving the key challenges of financial transparency and legally-binding online voting.

**A Comprehensive Management Ecosystem for Homeowners Associations (OSBB).**

Good Neighbor is a full-stack MVP designed to digitize community management in Ukraine. It simplifies communication between residents and board members while providing transparency in financial and administrative operations.

## Tech Stack
- **Frontend:** React.js, Tailwind CSS (Responsive Design)
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL (Relational Data Modeling)
- **Authentication & Infrastructure:** Google Firebase
- **Development Method:** AI-Assisted Orchestration (Vibe-coding for rapid prototyping)

---

## Key Features & User Roles

The platform implements a **3-tier access system**, ensuring tailored interfaces for different stakeholders:

### Resident Dashboard
- **Community Voting:** Secure participation in HOA decisions.
- **Financial Transparency:** Real-time balance checking and service payments.
- **Information Hub:** Access to house news, documents, and contacts.

### Board Chair (Admin) Panel
- **Management Suite:** Full control over house data and resident registers.
- **Voting Orchestration:** Create and monitor community polls.
- **Financial Oversight:** Manage payments and generate reports.

### Internal Platform Staff
- **System Audit:** Access to audit logs for monitoring database changes.
- **Infrastructure Management:** Direct database interaction for maintenance and troubleshooting.
